# LaslesVPN

This is a practice project built with:

- React JS
- Tailwind CSS

## Run Locally

Clone the project

```bash
  git clone https://github.com/theozebua/react-lasles-vpn.git
```

Go to the project directory

```bash
  cd react-lasles-vpn
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
